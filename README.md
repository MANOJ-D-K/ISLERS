ISLERS – Intelligent Micro-Steam Leak Detection & Energy Loss Monitoring System

Overview

ISLERS is an embedded system project developed to detect steam leaks in industrial pipelines. The system uses ESP32 to monitor temperature, pressure, and sound levels for identifying possible leaks and estimating energy loss. The collected data can be monitored in real time through Wi-Fi connectivity.

Project Objective

The objective of ISLERS is to detect steam leaks at an early stage using temperature, pressure, and acoustic sensing. The system helps reduce energy loss and improves industrial maintenance by providing continuous monitoring.

Project Features

• Detects steam leaks using multiple sensors.
• Monitors pipeline temperature and pressure.
• Identifies abnormal sound levels caused by leaks.
• Provides real-time monitoring through Wi-Fi.
• Designed with a custom PCB for industrial applications.

Hardware Used

• ESP32 Development Board
• PT100 RTD Sensor
• MAX31865 RTD Module
• Pressure Sensor
• MAX9814 Microphone
• OLED Display
• Custom PCB

Software Used

• Arduino IDE
• Embedded C

Communication

• Wi-Fi
